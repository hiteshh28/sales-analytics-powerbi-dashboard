# Sales Analytics Dashboard | Power BI

## Project Overview
This project presents an interactive sales analytics dashboard built using Power BI to analyze sales performance across categories, brands, and states.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Features
- KPI cards for Total Sales, Cost, Quantity, Transactions, and Profit
- Interactive filters for dynamic analysis
- Bar charts, pie charts, and map visualizations
